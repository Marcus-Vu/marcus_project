# HairsalonX — Website

Professional website for HairsalonX, a hair salon in Roermond, Netherlands.

## Stack
- **Framework:** Next.js 14 (App Router)
- **Styling:** Tailwind CSS
- **Language:** TypeScript
- **Deployment:** TBD (Vercel recommended)

## Pages
- `/` — Homepage (hero, USPs, about preview, services, reviews, CTA)
- `/over` — Over Josje (personal story, specializations)
- `/behandelingen` — Treatments & prices
- `/portfolio` — Gallery with Instagram integration
- `/contact` — Contact info, opening hours, booking CTAs

## Getting Started
```bash
cd hairsalonx
npm install
npm run dev
```
